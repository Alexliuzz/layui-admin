### 技术选型
-------------
Spring MVC、Mybatis、Mybatis-Plus(Mybatis增强插件)、Mysql、Maven、Layui

### 快速开始
-------------
1. git clone https://github.com/alexdoop/alex-admin.git
2. cd ~/alex-admin && mvn install
3. cd alex-admin-web && mvn jetty:run
4. http://localhost:8080
